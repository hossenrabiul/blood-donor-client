Terms and Conditions

Please review the following terms and conditions carefully.

1. Access Restrictions

If you are an unauthenticated user, you cannot access the full features of the web app. As an unauthenticated user, you will not be able to:

  • Search or filter blood requests, donors, or events.
  • Create new blood requests or make any updates to existing requests.

2. Blood Request Notifications

When a blood request is created, notifications will be sent if your blood type matches the request. Notifications will be updated instantly if a blood request is created and matches your blood type.

3. Features

As a registered user, you will have access to the following features:

  • Search all users and filter results by blood type, country, etc.
  • Search for all blood donation events and accept invitations to participate in them.
  • Donate your blood or request blood from others through your account.
  • Contact any user randomly in case of an urgent need for blood.

By using this web app, you agree to comply with the terms and conditions outlined above. We reserve the right to update or modify these terms at any time without prior notice.
